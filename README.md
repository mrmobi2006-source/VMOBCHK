## 📋 صيغة البطاقة

```
4815820235960174|06|30|123
CC|MM|YY|CVV
```

- **CC**: رقم البطاقة (13-19 رقم)
- **MM**: الشهر (01-12)
- **YY**: السنة (2 أو 4 أرقام)
- **CVV**: رمز الأمان

## 📊 النتائج المحتملة

- 🎉 **CHARGED** - تم الشحن (بطاقة صالحة تماماً)
- ✅ **APPROVED** - موافقة (بطاقة صالحة)
- ❌ **DECLINED** - مرفوضة
- ⚠️ **ERROR** - خطأ في الفحص
